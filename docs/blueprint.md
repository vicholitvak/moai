# **App Name**: HomeTaste

## Core Features:

- Cook Profile: Enable cooks to create a profile detailing their culinary expertise, specialties, and availability.
- User Profile: Allow users to create profiles with delivery addresses and payment methods.
- Dish Listing: Cooks can list dishes with descriptions, ingredients, preparation times, and prices.
- AI Delivery Estimation: AI tool analyzes cook availability, preparation times, delivery distance and current orders to provide a estimated delivery time to the clients
- Geolocation Services: Integrate a map that uses location services to find clients available near the cook.
- Payment Processing: Facilitate secure payment processing for orders.
- Push Notifications: Push notifications alert cooks of new orders and clients about delivery status

## Style Guidelines:

- Primary color: Earthy orange (#D2691E) to evoke feelings of warmth and home-cooked meals.
- Background color: Light beige (#F5F5DC), a desaturated hue analogous to the primary color, creating a comforting and clean backdrop.
- Accent color: Soft green (#8FBC8F) to add a fresh and natural contrast.
- Body font: 'Literata', a transitional serif giving a vintage and slightly formal feeling to the user, making them feel they are at home
- Headline font: 'Playfair', a modern sans-serif font used to denote elegance
- Use minimalist, line-based icons related to cooking, delivery, and user profiles.
- Subtle transitions and animations to provide feedback during order placement and status updates.
